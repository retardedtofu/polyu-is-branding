The Cell Code

A scannable mark drawn in the brand's own shapes, like Spotify Codes. The data lives in the shape and rotation of each cell, so colour stays free for the brand and the same code reads the same way in every palette. Each quarter's orientation carries 2 bits. The corner discs, each made of four quarters, act as finder patterns, and a CRC-16 checksum guards every read. Decoding runs entirely in your browser.

Decode

Pipeline: threshold → corner-disc detection → perspective correction (4-point homography) → per-cell ink-centroid classification → CRC-16 validation. It works flat-on or tilted and in mixed light; just fill at least a quarter of the frame with the code. The live status shows how many corner discs the scanner sees. Switch the palette and run the self-test again to confirm the read doesn't depend on colour. Camera frames stay local. For print where anyone needs to scan it, pair it with a standard QR; this code is meant for BDSIS surfaces and the BDSIS scanner.
