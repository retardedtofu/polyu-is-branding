Your Mark

Type anything — a name, a student number, a line of poetry, any language, any length. A 128-bit hash seeds a free cell composition on any grid up to 256 × 256: one lead family, sparse accent cells, and the quarter, split, circle and square primitives. Drag the preview's corner to resize it — the mark always fits, keeping its aspect ratio. Same input, same mark, forever. (The business-card application comes later — this is the raw generator.)

How it works: cyrb128 hashes the input to 128 bits; an sfc32 stream composes every cell in order — quarter 50%, split 16%, circle 12%, square 10%, empty 12%. Collision probability is negligible (2⁻¹²⁸-scale), and the mark is deterministic. Preview is canvas (65,536 cells render instantly); SVG download uses shared gradient definitions so files stay light; PNG caps at 8192 px.
