Appendix · The Cell System, applied

Built from the
logo's own grid

The monogram is a 3×3 grid: eight quarter-circle slices and one square split into two triangles. The Cell System turns that construction into the layout language for everything else — same cells, same primitives, same discipline. Pick a direction; every mockup re-themes live.

The four rules
Rule 1 · The cell

Every layout sits on a square module grid. Shapes, text blocks and images all snap to it.

Rule 2 · The vocabulary

Five primitives, all of the grid: quarter-disc, split square, circle, square, ghost cell. Nothing else.

Rule 3 · One corner
container

Every container — card, photo, slide block — takes exactly one quarter-circle corner. Never two, never four.

Rule 4 · One family

One colour family per piece. The full spectrum appears only in the logo and in monogram fragments.

Why this works better

The earlier comps decorated layouts with a gradient hairline; this system composes layouts from the monogram's own construction:

  • Cell bands and columns replace decorative bars — every shape sits in a real grid cell.
  • Ghost cells make the grid visible and give shapes room to breathe.
  • One rounded corner per container repeats the quarter-circle at every scale — cards, slides, the page itself.
  • Monogram fragments (multi-family cell clusters) are the only full-spectrum moments, reserved for hero surfaces like the poster band and card edge.
  • The "one cell different" motif — a lone vivid cell in a field of ghosts — is the brand idea itself: one student, one unique path.
