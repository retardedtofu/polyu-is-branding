## The spectrum (gradient pairs sampled from the logo)

The identity is fundamentally **gradient-based** — five hue families, each used as a gradient, reading as a full spectrum across the logo (left = warm "i", right = cool "S"):

| Family | Sampled stops | Role |
|---|---|---|
| **Purple** | `#634093` → `#8A467B` (toward red) | "i" top |
| **Red / Crimson** | `#ED174B` (≈ PolyU red territory) | "i" mid; accents, numbered headers in brochure |
| **Orange** | `#F36C3D` → `#F47C36` | "i" bottom; "BDSIS" text in lockup |
| **Green** | `#96C664` (light) → `#51BA51` → `#30B44D` | "S" top; Chinese name in lockup; section header bars |
| **Blue** | `#157AB5` / `#1080C0` → `#0766A0` (deep) | "S" bottom; English name text (~`#0080C0`); info panels |

Supporting neutrals: **dark grey** tagline text ~`#58595B`; white; the light **ice-blue** of the triangle-mosaic background (~`#D6ECF8` range).

## Observed color logic

- **Warm ↔ cool symmetry** mirrors the brand's two dimensions — human (warm) ↔ technology (cool) ([[brand-essence]]). Worth making explicit in the guideline.
- Gradients always run **within a hue family** (light→dark or hue-adjacent like purple→red); never across unrelated hues inside one shape.
- Large surfaces (brochure grid panels) use soft duotone gradients of one family; type reverses to white on them.
- Deep red/crimson doubles as the **editorial accent** (section numerals, "Interdisciplinary Core" headers) and bridges to the PolyU master-brand red ([[institutional-context]]).
